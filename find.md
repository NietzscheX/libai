``` shell
# find 命令是linux里面常用的命令
# 在决定备份文件保留时间的时候尤为有用
# 但是在mtime 又分为+ - 不带符号,傻傻的分不清楚
# 对于访问时间mtime的使用,参考下面的时间轴,
# 过去的7天里,和现在/未来
#   7__6__5__4__3__2__1__现在__明天__后天__...未来
#mtime不能感知未来,只能根据过去已经发生的岁月痕迹进行查询^_^

# 所以说  
  -mtime  1  代表恰好是昨天发生的故事
  -mtime  2  代表恰好是前天发生的故事
  -mtime  +3   这里的+ 表示从3之外,开始向坐标轴的负半轴移动,也就是包括3,4,5,6,7...
  -mtime  -3   这里的- 表示3天之内,包括3 2 1
  这里可以形象的记忆,我给你3天时间,那么是-3 -2 -1 ...这样时间一天天减少
                      3天前的数据当然包括更早的4天前,5天前,不断的加呗.
  find -name +90 -exec rm -rf {} \;
  当然可以简化一下
  find -name +90 -delete
```
